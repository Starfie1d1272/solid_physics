## 2. 作业目标（题目一）
本题基于 ObsPy 的 TauP 工具，围绕火星内核结构开展射线追踪与震相走时分析，目标如下：
1. 构建两套 1D 火星速度模型（无固态内核 / 有固态内核）。
2. 在震源深度 33 km、震中距 27° 下绘制两模型射线路径图（对应文献 Fig1a/Fig1b 对比思路）。
3. 对比两模型可见震相与走时差，识别固态内核影响。

## 3. 数据与方法（题目一）
### 3.1 参考文献
- Stahler et al., Nature (2025)
- https://www.nature.com/articles/s41586-025-09361-9

### 3.2 工具与环境
- Python 3.x
- ObsPy (`obspy.taup`, `obspy.taup.taup_create`)
- NumPy / Pandas / Matplotlib

### 3.3 参数设置

- 震源深度：33 km
- 主对比震中距：27°
- 模型 A：无固态内核
- 模型 B：有固态内核（ICB 深度约 2776.5 km）

### 3.4 处理流程
1. 编写两套 `.nd` 速度模型并编译为 TauP `.npz`。

3. 绘制 27° 条件下的双图对比。
4. 输出 27° 下无核/有核两张震相走时表并做文字比较。

## 5. 讨论（题目一）
### 5.1 物理解释
结果显示，引入固态内核后，深部射线传播路径与可见震相集合发生变化；在 27° 条件下，有核模型出现了无核模型中不存在的深部相分支。

### 5.2 与文献差异来源
1. 本作业使用参数化 1D 模型，并非文献反演的完整模型（查找了GitHub源代码发现GD.nd文件缺失）。
2. 外核/内核速度梯度与分层细节做了简化。
3. 震中距与相位筛选范围有限，会影响分支可见性。

### 5.3 关键调试记录
1. `.nd` 模型缺少 `outer-core` / `inner-core` 会导致 TauP 解析失败。
2. 复杂深部相需在相位池中显式加入，否则默认搜索可能缺失。
3. 射线绘图采用笛卡尔坐标转换，避免极坐标插值带来的轨迹畸变。
4. 优弧镜像仅用于可视化排版，不改变物理计算结果。
5. `PKPPKP` 采用“先行程后深度”的两阶段分类以稳定区分 `n/ab/bc`。
