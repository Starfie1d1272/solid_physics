## 题目一：火星内核 TauP 射线追踪

### 具体要求
1. 构建两套火星 1D 速度模型并编译为 TauP 模型。
2. 在震源深度 33 km、震中距 27° 下绘制射线路径图（对应 Fig1a/Fig1b 对比思路）。
3. 分析受固态内核影响的震相，并比较走时差异。
4. 给出结果解释，说明与 Nature 文献图的异同。

### 结果摘要（27°）
- 无核模型可见分支：`PKKP`, `P'P'n`, `P'P'r_ab`, `P'P'r_bc`
- 有核模型可见分支：`PKiKP`, `PKIIKP`, `PKKP`, `P'P'n`, `P'P'r_ab`, `P'P'r_df`
- 共有分支：`PKKP`, `P'P'n`, `P'P'r_ab`
- 仅无核出现：`P'P'r_bc`
- 仅有核出现：`PKiKP`, `PKIIKP`, `P'P'r_df`

### 调试记录
1. `.nd` 模型需严格标注 `outer-core` / `inner-core`，否则 TauP 可能无法正确解析核边界。
2. 复杂深部相需要在相位池中显式加入高阶相名，否则默认搜索可能缺失。
3. 射线绘图采用笛卡尔坐标转换而非直接极坐标插值，以避免轨迹畸变。
4. 为对齐文献版式，对优弧分支做镜像展示（仅影响可视化，不影响物理结果）。
5. `PKPPKP` 使用两阶段分类（先行程后深度）以稳定区分 `n/ab/bc`。

## 题目二：直达 P 波射线路径与展平变换验证

### 具体要求
1. 基于 IASP91 模型，实现直角坐标 / 球坐标 / 展平变换三种射线追踪算法。
2. 在震源深度 10 km、震中距 5° 和 45° 下计算射线路径与走时。
3. 对比三种方法的走时残差，验证 EFT 的数学等价性。
4. 分析直角坐标平面假设的适用范围与误差来源。

### 核心方法
- 采用换元积分法（$u = \sqrt{r - r_{\text{turn}}}$）消除积分奇点。
- 打靶法通过暴力搜索 + Brentq 求根确定出射角。
- 展平变换公式：$z_f = R \ln(R/r)$, $v_f = (R/r) \cdot v$

### 结果摘要
- 5° 近震：EFT 与球坐标残差 0.0002 s，直角坐标残差 0.3766 s（曲率影响可忽略）
- 45° 远震：EFT 与球坐标残差 0.1296 s（数值积分误差），直角坐标残差 47.86 s（曲率效应显著）
- 路径对比图显示 EFT 与球坐标路径完美重叠，直角坐标路径严重偏离
