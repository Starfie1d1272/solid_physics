# Homework 1: 地震频度与 Gutenberg-Richter 定律分析

> **Due Date:** 2026/03/09

## 📝 任务概述 (Assignment Objective)
本作业旨在基于真实地震目录数据，验证并拟合经典地震学经验公式——**古登堡-理查特（Gutenberg-Richter）定律**。

### 具体要求：
1. **数据获取：** 从 USGS 网站下载 PDE 地震目录。
   * **时间跨度：** 2010 ~ 2020 年
   * **震级范围：** $M \ge 5.0$
2. **数据统计：** 以 0.5 级为一个统计区间（Bin），统计每年各区间的地震频次。
3. **模型拟合：** 在半对数坐标系下绘制震级-频度关系图，并进行线性拟合，求解 G-R 关系式中的常量 $a$ 和 $b$：
   $$ \log_{10} N = a - bM $$
4. **拓展分析：** 对计算得出的 $b$ 值进行物理意义上的探讨与拓展。

## 📁 文件结构 (File Structure)
* `data/query.csv`: 从 USGS 下载的原始地震目录数据。
* `analysis.py`: 核心数据清洗、分箱 (Binning) 统计与线性回归计算代码。
* `figures.py` / `figures/`: 包含最终生成的拟合散点图与回归直线。
* 📄 **`HW1_Report.pdf`**: 包含完整的数据分析、拟合结果图表及详细的物理意义讨论报告。
